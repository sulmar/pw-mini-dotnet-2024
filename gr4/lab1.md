- wskazówka: **Wyłącz alias w PowerShell**.
PowerShell domyślnie używa aliasu curl dla Invoke-WebRequest. 
Aby wyłączyć ten alias w bieżącej sesji PowerShell, możesz użyć poniższej komendy:

```bash
Remove-Item Alias:curl	  
```
